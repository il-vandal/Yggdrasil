# Yggdrasil
Develop a fully fledged Microservice environment based on best practices including:
- API Gateway (Node.js)
- Multiple Microservices (C#) contained in Docker containers
- SQL Server DB per Microservice
- Cachin DB (Redis)
- Distributed Tracing
- Centralized Logging
- Event Bus
- Service Discovery
- Load Balancing microservices
